# **Clock App**
An aesthetic clock app by Silas Lundquist.

## Features
- Light and Dark Background Themes
- Unique Text themes
- Fullscreen mode
- 12h or 24h mode
- Option for seconds

## Coming Features
- Background and text theme selector
- Custom background theme maker
- Timer and alarm
